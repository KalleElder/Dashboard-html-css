# Dashboard-html-css
Utbildnings uppgift 2


Crypto dApp Dashboard (HTML + CSS)

Det här är min enkla, responsiva krypto-dashboard byggd med HTML och CSS (ingen JavaScript). Jag använder demo-data, ankarlänkar och stöd för mörkt/ljust läge.

Vad jag har byggt

Meny med ankare: Markets, Heatmap, Wallet, Transactions, Create

Markets: flera coins (BTC, ETH, SOL, ADA, DOGE, AVAX, MATIC) med mini-grafer (linje + staplar)

Heatmap: färgplattor som visar 24h-förändring

Wallet: balans, adress och nyckeltal

Transactions: tabell med zebra/hover

Create: formulär med flera fält och tydliga focus-stilar


Öppna index.html i webbläsaren

Tips: Jag testar responsivitet och mörkt/ljust läge i DevTools.
